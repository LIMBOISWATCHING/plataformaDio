# EdF-A.I

Assistente Financeiro Inteligente desenvolvido com Streamlit e Google Gemini.

## Funcionalidades

* Diagnóstico financeiro personalizado
* Análise de renda e despesas
* Score financeiro
* Planejamento de metas
* Chat financeiro com IA
* Histórico de conversas

## Tecnologias

* Python
* Streamlit
* Google Gemini API
* Pandas

## Como executar

Instale as dependências:

pip install -r requirements.txt

Crie um arquivo .env:

GOOGLE_API_KEY=sua_chave

Execute:

streamlit run app.py

## Objetivo

Auxiliar usuários a melhorar sua organização financeira através de Inteligência Artificial Generativa.
